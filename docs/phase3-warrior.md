# Phase 3: Warrior — Replace Every Google App

> *"Every app you degoogle is a chain you break."*

## The Google Dependency Map

Before you start, understand what Google has on you. Go to [myaccount.google.com/dashboard](https://myaccount.google.com/dashboard) and see the scale of data they hold.

Then go to [takeout.google.com](https://takeout.google.com) and request a full export. You'll need this data to migrate.

## Priority Order

Don't try to replace everything at once. Follow this order — highest impact first:

### 1. Email (Critical)

**Gmail → ProtonMail**

Gmail reads every email to serve ads (they stopped scanning for ads in 2017, but still process email for "features"). Your email contains: bank statements, medical records, passwords, conversations.

| Option | Price | Storage | Notes |
|--------|-------|---------|-------|
| **ProtonMail** | Free / €4/mo | 1-15GB | Swiss law, E2E encryption, open source |
| **Tuta** (Tutanota) | Free / €3/mo | 1-10GB | German law, E2E, calendar included |
| **Disroot** | Free | 1GB | Community-run, Netherlands |
| **Self-hosted (Phase 4)** | VPS cost | Unlimited | Full control via Stalwart Mail |

**Migration steps:**
1. Sign up at [proton.me](https://proton.me)
2. Use ProtonMail's Easy Switch to import Gmail emails
3. Set up forwarding: Gmail → Settings → Forwarding → add ProtonMail address
4. Update important accounts (bank, government, etc.) to your new email
5. Keep Gmail forwarding for 30-60 days to catch stragglers
6. After 60 days: turn off forwarding, archive Gmail

**Pro tip:** Use email aliases (SimpleLogin, included in Proton Unlimited) to give each service a unique address. If one leaks, you know who sold your data.

### 2. Passwords (Critical)

**Chrome Passwords → Bitwarden**

Chrome passwords are synced to Google's servers. If Google is compromised, so are all your passwords.

| Option | Price | Notes |
|--------|-------|-------|
| **Bitwarden** | Free / $10/yr | Open source, cross-platform, best all-around |
| **KeePassXC** | Free | Offline-only, file-based (no cloud sync) |
| **Vaultwarden (Phase 4)** | VPS cost | Self-hosted Bitwarden server |

**Migration steps:**
1. **Export from Chrome:** Settings → Passwords → ⋮ → Export passwords (CSV)
2. **Import to Bitwarden:** [bitwarden.com](https://bitwarden.com) → Create account → Tools → Import → Chrome CSV
3. **Install Bitwarden extension** in Firefox
4. **Install Bitwarden app** on phone
5. **Delete Chrome passwords:** Settings → Passwords → delete all
6. **Disable Chrome password saving:** Settings → Passwords → Offer to save → OFF

**Then: change your most important passwords** (email, bank, social media) to strong, unique ones generated by Bitwarden.

---

### 3. Search (Easy, High Impact)

**Google Search → DuckDuckGo / Brave Search / SearXNG**

Every Google search is logged, profiled, and used for advertising.

| Option | Privacy | Quality | Notes |
|--------|---------|---------|-------|
| **DuckDuckGo** | Good | Good | No tracking, based on Bing index |
| **Brave Search** | Great | Great | Independent index, no Big Tech |
| **SearXNG (Phase 4)** | Perfect | Great | Self-hosted metasearch, zero logging |
| **Startpage** | Good | Great | Google results without tracking |

**How to switch:**
1. Firefox → Settings → Search → Default Search Engine
2. Phone: change default browser search engine
3. That's it. You'll adapt in 2-3 days.

**If results aren't good enough:** Add `!g` to any DuckDuckGo search to get Google results (through DuckDuckGo's proxy).

### 4. Messaging (Important)

**WhatsApp/Google Messages → Signal**

WhatsApp is owned by Meta. Google Messages reads your SMS. Both collect metadata (who you talk to, when, how often).

| Option | E2E Encrypted | Metadata | Notes |
|--------|--------------|----------|-------|
| **Signal** | Yes (default) | Minimal | Gold standard, open source |
| **Element (Matrix)** | Yes (opt-in) | Self-hosted possible | Federated, Phase 4 has your own server |
| **Session** | Yes | Zero metadata | No phone number needed, uses onion routing |
| **SimpleX** | Yes | Zero metadata | No identifiers at all |

**Migration steps:**
1. Install [Signal](https://signal.org/) on phone and desktop
2. Tell your close contacts: "I'm moving to Signal for privacy"
3. Set Signal as default SMS app (Android) — it handles both
4. Create a Signal group for family/friends
5. After 30 days: reduce WhatsApp usage, eventually delete

**Reality check:** You won't get everyone to switch. That's OK. Use Signal for people who care, keep WhatsApp for the rest, but don't share sensitive info there.

### 7. Two-Factor Authentication

**Google Authenticator → Aegis / Ente Auth**

Google Authenticator now syncs to Google's cloud (enabled by default). Your 2FA codes shouldn't be in the same place as your email.

| Option | Platform | Notes |
|--------|----------|-------|
| **Aegis** | Android | Open source, encrypted backup, best on Android |
| **Ente Auth** | Android/iOS/Web | E2E encrypted, cross-platform |
| **KeePassXC** | Desktop | Built-in TOTP, offline |

**Migration steps:**
1. Install Aegis/Ente Auth
2. For each service with 2FA: disable old → re-enable with new app
3. **Export Aegis backup** (encrypted) and save it securely
4. Delete Google Authenticator

### 11. Phone (Advanced)

**Stock Android → GrapheneOS**

This is the nuclear option. GrapheneOS removes all Google services from your Pixel phone.

**Requirements:** Google Pixel phone (5 or newer recommended)

**What you get:**
- Zero Google services on your phone
- Sandboxed Google Play (optional — run Google apps in an isolated sandbox)
- Monthly security updates
- Best mobile security/privacy available

**Installation:** [grapheneos.org/install](https://grapheneos.org/install) — web-based installer, takes 15 minutes.

**Not ready for GrapheneOS?** Alternatives:
- **LineageOS** — degoogled Android for many phone brands
- **CalyxOS** — privacy-focused, with microG (partial Google compatibility)
- **Just uninstall Google apps** — disable what you can on stock Android

## Google Data Export & Deletion

### Export everything first

1. Go to [takeout.google.com](https://takeout.google.com)
2. Select **all services**
3. Choose: Export once → .zip → 2GB files
4. Wait for email with download links
5. Download and verify everything

### Delete service by service

After migrating, delete data from each service:
- **Gmail:** Select all → Delete → Empty trash
- **Drive:** Select all → Delete → Empty trash
- **Photos:** Select all → Delete → Empty trash
- **Keep:** Delete all notes
- **Calendar:** Delete all calendars

### Delete Google Account (optional, final step)

Only after Phase 4-5 when everything is migrated:
1. [myaccount.google.com/delete-account](https://myaccount.google.com/delete-account)
2. Download your data one last time
3. Confirm deletion

**Warning:** This is irreversible. Make sure you've migrated everything.
